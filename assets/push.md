## Отправка изменений в репозиторий

Команда **git push** - отправит подготовленный "пакет" изменений в систему контроля версий.

Чтобы добавить все (кроме игнорируемых) файлы из каталога в индекс использовать команду:
> ```bash=
> git push
> ```
